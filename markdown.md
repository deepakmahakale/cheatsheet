# Markdown

### Console output 

- Instead of `shell`, `bash` or `sh`:

```shell
[Deepaks-MacBook-Pro:~] deepak% cd workspace/
[Deepaks-MacBook-Pro:~/workspace] deepak% bundle install
```

- Use `console` for a snippet with this output:

```console
[Deepaks-MacBook-Pro:~] deepak% cd workspace/
[Deepaks-MacBook-Pro:~/workspace] deepak% bundle install
```
